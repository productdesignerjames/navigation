# Navigation Design Implementation

This project implements the navigation and hero section from the provided Figma design using plain HTML, CSS, and JavaScript.

## Files
- `index.html`: Main HTML structure
- `styles.css`: Styling for layout and components
- `script.js`: Placeholder for interactivity

## How to Run
Open `index.html` in your browser to view the design.

## Customization
- Update `styles.css` for style changes
- Add interactivity in `script.js`

---

This project was scaffolded and managed with GitHub Copilot in VS Code.
